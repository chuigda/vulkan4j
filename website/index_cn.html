vulkan4j

Vulkan 的 Java 绑定，使用 Project Panama API

vulkan4j 是什么

vulkan4j 是一个 Java 库，提供了对下一代图形与计算 API Vulkan 的访问。这些功能基于 Project Panama API，因而非常高效，也符合 Java 平台的发展趋势。API 以类型安全且用户友好的方式进行了封装，使其适应 Java 生态系统。

vulkan4j 是开源软件，并且不收取任何费用。
